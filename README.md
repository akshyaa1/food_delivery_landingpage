# QuickBite - Food Delivery Landing Page

A sleek, responsive, modern food delivery landing page built using **Bootstrap 5** and **Bootstrap Icons**. This project features a fully responsive navigation bar, a high-impact hero banner, a features grid, and an interactive popular menu display.

## 🚀 Features
* **Fully Responsive Design:** Adapts smoothly to mobile, tablet, and desktop viewports.
* **Modern Hero Banner:** Includes visually engaging overlay text and clear Call-to-Action (CTA) elements.
* **Structured Sections:** Cleanly divided into Features, Popular Dishes, and Footer sections.
* **Optimized Delivery UI:** Built using lightweight Bootstrap CDN components for lightning-fast load times.

## 🛠️ Tech Stack
* **HTML5:** Semantic layout structure.
* **CSS3:** Custom linear-gradients and spacing overrides.
* **Bootstrap 5 (CDN):** Component architecture and responsive grids.
* **Bootstrap Icons:** Vector graphics for service indicators.
